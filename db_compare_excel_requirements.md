# 数据对比结果汇总（Excel输出）需求说明书

## 1. 背景

当前系统已经生成约 14000 条数据库结构对比明细数据，但数据粒度过细，无法直接用于汇报或决策分析。

需要对这些原始数据进行统计汇总，并输出为 Excel 报表，用于：

- 向领导汇报
- 风险评估
- 迁移进度跟踪

---

## 2. 目标

生成一份 Excel 汇总文件，包含：

1. 总体统计
2. 差异分类统计
3. 风险分级统计
4. 明细数据（可选）

---

## 3. 输入数据结构（假设）

原始数据字段示例：

| 字段名 | 说明 |
|--------|------|
| source_db | 源库 |
| source_schema | 源 schema |
| table_name | 表名 |
| column_name | 字段名 |
| diff_type | 差异类型 |
| detail | 详细描述 |

---

## 4. 输出文件说明

输出文件：

compare_summary.xlsx

包含多个 Sheet。

## 6. 统计规则

### 表级统计

- 以 table 为单位
- 有任意差异即算问题表

### 差异分类

- MISSING_COLUMN
- TYPE_MISMATCH
- LENGTH_MISMATCH
- DEFAULT_MISMATCH

### 风险分类

| 条件 | 风险 |
|------|------|
| 无差异 | 低 |
| 长度/nullable差异 | 中 |
| 缺字段/类型不一致 | 高 |

---

## 7. 技术要求

- Java 实现
- 支持 1万+ 数据
- Excel 可直接打开

---

## 8. 验收标准

- 数据正确
- Excel 完整
- 性能可接受

---

## 9. 核心需求

将原始对比明细数据，汇总成多维度统计结果，并输出为 Excel。
